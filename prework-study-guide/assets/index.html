Git

  • Git is an online version control service that connects with GitHub to allow for multi person / team working.
  • GitHub is a linked online community where repositories are created to work on within teams. Within repositories, issues are created to work on sections of code.
  • Version control is a very important part of web development as if mistakes are made that cannot be undone, a previous version can be accessed to redo and continue.
  • The following codes can be used within the Command Line Interface:
  • git status: checks what branch we are currently on
  • git checkout -b "branch-name": creates a new branch and switches to it
  • git merge "branch-name": merges branch with main branch
  • git clone "repo-name": clones repo from GitHub
